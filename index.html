This is a universal deep link redirector for the SpeedTest app. It allows you to create shareable links that will automatically open specific locations in the SpeedTest mobile application.

What is a Deep Link?

A deep link is a URL that opens a specific page or location within a mobile application, rather than just launching the app's home screen. This redirector helps bridge web URLs to app-specific content.

How to Use

To create a deep link that opens a specific location in the SpeedTest app, use the following URL structure:

https://armandfardeau.github.io/DeepLinkRedirect/deeplink.html?to=speedtest://location

URL Parameters

  • to - (Required) The deep link URL that should be opened in the SpeedTest app

Example Usage

Basic Deep Link

Open a specific server in SpeedTest:

https://armandfardeau.github.io/DeepLinkRedirect/deeplink.html?to=speedtest://server/12345

Custom Action

Start a speed test immediately:

https://armandfardeau.github.io/DeepLinkRedirect/deeplink.html?to=speedtest://start
Note: The user must have the SpeedTest app installed on their device for deep links to work. If the app is not installed, the behavior will depend on the device's operating system (typically showing an error or redirecting to the app store).

How It Works

  1. The user clicks on the redirector link
  2. The deeplink.html page extracts the to parameter from the URL
  3. The page attempts to open the deep link using JavaScript
  4. If successful, the SpeedTest app opens to the specified location
  5. If the app is not installed, the user will be notified

Technical Details

This redirector is a static HTML-based solution that uses JavaScript to handle the deep link redirection. It's hosted on GitHub Pages and requires no backend server.
